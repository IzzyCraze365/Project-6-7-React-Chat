# Testing Code

## Get Server started

1. Open `server` Folder
2. Open `package.json` in terminal Window
3. Type `nodemon`

## Get Client Website started

1. Open `client` Folder
2. Open `package.json` in terminal Window
3. Type `npm start`

## Getting Started

`npx create-react-app my-app`
`cd my-app`
`npm start`
